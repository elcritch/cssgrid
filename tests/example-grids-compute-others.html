Notes on Position Calculation:

The CSS Grid calculates track positions as follows:

  • Column 1: Starts at 0px, width 40px
  • Column 2: Starts at 50px (40px + 10px gap), width 50px
  • Column 3: Starts at 110px (50px + 50px + 10px gap), width auto (takes remaining space)
  • Column 4: Starts at 900px, width 50px
  • Column 5: Starts at 960px (900px + 50px + 10px gap), width 40px
  • End: Starts at 1000px (960px + 40px + 0px gap, no gap after final track)
