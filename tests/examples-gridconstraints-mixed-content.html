How Mixed Content Sizing Works:

This example demonstrates how a CSS grid with mixed column sizing distributes space:

  • min-content: Takes the minimum width needed for content (50px)
  • auto: When used with fractional units, takes only the space needed for content (75px)
  • 1fr: Takes all remaining space (375px)

The grid template columns definition:

grid-template-columns: min-content auto 1fr;

Key behavior: When auto is used with fr units, the auto column takes just enough space for its content, and any remaining space goes to the fractional column(s).

Nim Test Equivalent:

parent.gridTemplate.lines[dcol] = @[
  initGridLine(csContentMin()),
  initGridLine(csAuto()),
  initGridLine(csFrac(1))
]

// With child elements of widths 50px, 75px, and 100px minimum
// In 500px container, the resulting widths are:
// - Column 1: 50px (min-content)
// - Column 2: 75px (auto)
// - Column 3: 375px (remaining space)
